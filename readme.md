# MiniStash

A **Mini Stash** for all your files

### Features

- [x] view files
- [x] open folders from current dir
- [x] create new files
- [x] create new folders
- [x] download files
- [x] open folders by clicking files view
- [ ] proper theme support
- [ ] delete files
- [ ] rename files
- [ ] login
- [ ] rate-limits
- [ ] server api auth for login
